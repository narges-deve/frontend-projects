# BLAXE – Login & Signup UI

A modern responsive Login & Signup user interface built with HTML and CSS.

This project is part of my frontend learning journey where I practice building real-world UI components inspired by modern design systems.

---

## ✨ Features

- Modern split layout design (Login + Info section)
- Responsive design for mobile and desktop
- Animated background circles (CSS-only)
- Clean and minimal form UI
- Social media icons integration (Font Awesome)
- Input focus effects for better UX
- Modern color palette and typography

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Media Queries)
- Font Awesome Icons

---

## 📱 Responsive Design

This UI is fully responsive and adapts to:
- Desktop screens
- Tablets
- Mobile devices

---

## 🎯 Purpose

The goal of this project is to improve my frontend skills by recreating real-world UI designs and understanding layout structure, spacing, and responsiveness.
